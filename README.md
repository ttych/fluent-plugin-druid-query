# fluent-plugin-druid-query

[Fluentd](https://fluentd.org/) plugin to query druid.


## Installation

### RubyGems

```
$ gem install fluent-plugin-druid-query
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-druid-query"
```

And then execute:

```
$ bundle
```
